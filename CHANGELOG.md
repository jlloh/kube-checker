## [0.2.0] - 2021-04-06
### Added
* Added clap to accept command line arguments to control behaviour
  * --disable-filter
  * --generate-csv
  * --print-table
* Added option to disable filtering to print out all pod information
* Added some metrics for CPU request, replicas, etc.
